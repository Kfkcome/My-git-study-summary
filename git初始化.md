# GIT初始化
usage: git init [-q | --quiet] [--bare] [--template=<template-directory>] [--shared[=<permissions>]] [<directory>]

    --template <template-directory> 目录使用模板目录
    --bare                			创建一个简单的存储库
    --shared[=<permissions>] 		指定git存储库将在多个用户之间共享
    -q, --quiet           			静默
    --separate-git-dir <gitdir> 	从工作树中分离git目录
